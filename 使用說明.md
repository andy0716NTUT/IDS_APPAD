# IDS_APPAD 使用說明

> 自適應隱私保護異常檢測系統

## 🚀 快速開始

### 1. 安裝依賴
```bash
pip install -r requirements.txt
```

### 2. 啟動系統
```bash
# 一鍵啟動 (自動訓練模型)
python run.py

# Mock模式 (無需訓練,快速測試)
python run.py --mock-mode
```

### 3. 訪問API
- 文檔: http://localhost:8000/docs
- 健康檢查: http://localhost:8000/health

## 📦 專案結構

```
IDS_APPAD/
├── project1_classifier/    # 隱私分類器
├── project2_encryption/     # 加密檢測
├── project3_core/          # 系統核心 (REST API)
├── tests/                  # 測試
├── examples/               # 範例
├── run.py                  # 啟動腳本
└── verify.py              # 驗證腳本
```

## 🧪 測試

```bash
# 快速驗證
python verify.py

# 完整測試
python tests/test_integration.py

# 單獨測試各項目
python project1_classifier/classifier.py
python project2_encryption/detector.py
```

## 📝 API 使用範例

### 使用 cURL
```bash
curl -X POST http://localhost:8000/detect \
  -H "Content-Type: application/json" \
  -d '{
    "payload": "username=admin&password=secret",
    "source_ip": "192.168.1.100",
    "destination_ip": "203.0.113.50"
  }'
```

### 使用 Python
```python
import requests

response = requests.post('http://localhost:8000/detect', json={
    'payload': 'POST /login {"password":"secret"}',
    'source_ip': '192.168.1.100',
    'destination_ip': '203.0.113.50'
})

result = response.json()
print(f"異常: {result['is_anomaly']}")
print(f"敏感: {result['is_sensitive']}")
print(f"方法: {result['detection_method']}")
```

### 使用範例客戶端
```bash
python examples/client.py
```

## 🎯 系統運作方式

```
流量輸入 → 隱私分類 → 路由決策
                ↓
        ┌───────┴────────┐
    敏感流量          一般流量
        ↓                ↓
    加密檢測          明文檢測
    (~200ms)          (~5ms)
        ↓                ↓
        └───────┬────────┘
                ↓
            返回結果
```

## 📊 性能指標

- 明文檢測: <1ms
- 加密檢測: ~210ms
- 準確度: 100%

## 🔧 常用命令

```bash
# 啟動選項
python run.py --host 0.0.0.0 --port 8080  # 自定義主機端口
python run.py --mock-mode                  # Mock模式
python run.py --force-train                # 強制重新訓練

# 查看統計
curl http://localhost:8000/metrics

# Mock測試
curl -X POST http://localhost:8000/test/mock
```
